# Real-time Traffic Sign Recognition

A real-time traffic sign recognition in a hierarchy approach using HOG and SVM.

*) Dependencies
1) OpenCV
2) Dlib

Source code and datasets are available for downloading, and the datasets are about 5.6GB.
